# 9. 测试重点清单

|测试项|检查点|
|---|---|
|资料上传|是否支持规定格式，失败提示是否清晰|
|资料解析|是否能生成基本结构和知识点|
|计划生成|是否生成合理任务，时长是否超预算|
|任务学习|资料是否匹配任务，问答是否有引用|
|测验生成|题目是否与任务相关|
|自动评分|得分逻辑是否稳定|
|复习回流|完成后是否按规则再次出现|
|动态调整|未完成任务是否正确顺延|

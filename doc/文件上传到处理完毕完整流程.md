## 第二阶段：文件保存

### 步骤 2.1：生成唯一文件名

```python
# api/routers/sources.py

# 防止文件名冲突
file_path = generate_unique_filename(upload_file.filename, UPLOADS_FOLDER)
# 例如: "document.pdf" → "/uploads/document.pdf"
# 如果已存在 → "/uploads/document (1).pdf"
```

**文件名冲突处理**：

```
用户上传 file.pdf（第一次）
    ↓
保存到 /uploads/file.pdf

用户再上传 file.pdf（第二次）
    ↓
检测到已存在
    ↓
保存到 /uploads/file (1).pdf

用户再再上传 file.pdf（第三次）
    ↓
检测到 file (1).pdf 已存在
    ↓
保存到 /uploads/file (2).pdf
```

### 步骤 2.2：写入磁盘

```python
# 保存到 /uploads 目录
file_path = save_uploaded_file(upload_file)

# 内部实现：
with open(file_path, "wb") as f:
    content = await upload_file.read()  # 异步读取
    f.write(content)  # 写入磁盘
```

### 步骤 2.3：失败时清理

```python
except Exception:
    if os.path.exists(file_path):
        os.unlink(file_path)  # 删除已写入的部分文件
    raise
```

### 步骤 2.4：验证笔记本存在

```python
# 确保关联的笔记本存在
for notebook_id in source_data.notebooks:
    notebook = await Notebook.get(notebook_id)
    if not notebook:
        raise HTTPException(status_code=404, detail="Notebook not found")
```

## 第四阶段：提交异步任务

### 步骤 4.1：准备 content_state

```python
# 传递给 Worker 的内容信息
content_state = {
    "file_path": "/uploads/document.pdf",
    "delete_source": False
}
```

### 步骤 4.2：构建任务参数

```python
# SourceProcessingInput 定义在 commands/source_commands.py
command_input = SourceProcessingInput(
    source_id=str(source.id),           # "source:abc123"
    content_state=content_state,         # {"file_path": "..."}
    notebook_ids=source_data.notebooks,  # ["notebook:abc123"]
    transformations=[],                  # 用户选择的转换
    embed=True,                        # 是否生成嵌入
)
```

### 步骤 4.3：提交到队列

```python
# api/command_service.py
command_id = await CommandService.submit_command_job(
    "open_notebook",        # 应用名
    "process_source",       # 命令名
    command_input.model_dump(),
)
# 返回: "cmd_xyz789"
```

### 步骤 4.4：更新来源记录

```python
# 关联命令 ID，便于后续查询状态
source.command = ensure_record_id(command_id)
await source.save()
```

### 步骤 4.5：立即返回响应

```python
# 用户不用等待处理完成
return SourceResponse(
    id=source.id,
    title="Processing...",      # UI 显示处理中
    command_id=command_id,       # 用于轮询状态
    status="new",
    processing_info={"async": True, "queued": True},

```

## 第六阶段：LangGraph 工作流

### 节点 1：content_process - 提取内容

```python
# open_notebook/graphs/source.py

# 1. 配置提取参数
content_state["document_engine"] = "auto"  # 自动选择引擎
content_state["output_format"] = "markdown"  # 输出格式

# 2. 使用 content-core 提取
processed_state = await extract_content(content_state)
# 提取 PDF/网页/音视频的文本内容

# 3. 返回提取结果
return {"content_state": processed_state}
# 包含: content, title, url, file_path
```

| 输入类型 | 处理方式 |
|---------|---------|
| 文件 (PDF, DOCX...) | 使用 `content-core` 库提取 |
| URL 链接 | 抓取网页内容 |
| YouTube 视频 | 尝试获取字幕 |
| 纯文本 | 直接使用 |

### 节点 2：save_source - 保存来源

```python
# 1. 获取来源记录
source = await Source.get(source_id)

# 2. 更新内容
source.full_text = processed_state.content  # 提取的文本
source.title = processed_state.title or "我的论文"
source.asset = Asset(file_path="/uploads/document.pdf")

# 3. 保存到数据库
await source.save()

# 4. 提交嵌入任务（异步）
if embed:
    await source.vectorize()
# 内部: submit_command("open_notebook", "embed_source", {...})
```

### 条件边：trigger_transformations

```python
# 检查是否有转换
if len(apply_transformations) > 0:
    # 为每个转换创建并行任务
    return [Send("transform_content", {...}) for t in transformations]
else:
    return []  # 直接结束
```

### 节点 3：transform_content - 转换内容

```python
# 对每个转换执行 AI 分析

# 1. 调用转换图
result = await transform_graph.ainvoke({
    "input_text": source.full_text,
    "transformation": transformation
})
# AI 生成: 摘要/洞察/QA 等

# 2. 保存洞察
await source.add_insight(
    title=transformation.title,
    content=result["output"]
)
```

## 第七阶段：嵌入任务

### 提交嵌入任务

```python
# open_notebook/domain/notebook.py
async def vectorize(self) -> str:
    command_id = submit_command(
        "open_notebook",
        "embed_source",
        {"source_id": str(self.id)},
    )
    return command_id
```

### Worker 执行嵌入

```python
# commands/embedding_commands.py
@command("embed_source", ...)
async def embed_source_command(input_data):
    # 1. 获取来源文本
    source = await Source.get(input_data.source_id)

    # 2. 文本分块
    chunks = chunk_text(source.full_text)
    # 默认: 1500 字符/块，225 字符重叠

    # 3. 批量生成嵌入
    embeddings = generate_embeddings(chunks)

    # 4. 保存到数据库
    for chunk, embedding in zip(chunks, embeddings):
        await repo_create("source_embedding", {
            "source": source.id,
            "content": chunk,
            "embedding": embedding
        })
```

## 同步 vs 异步处理对比

| 特性 | 同步 | 异步 |
|------|------|------|
| 用户等待 | 约 2-5 分钟 | 立即返回 |
| 响应内容 | 完整来源 | 只有 command_id |
| 轮询 | 不需要 | 需要 |
| 超时风险 | 有（5分钟） | 无 |
| 适用场景 | 小文件/文本 | 大 PDF/视频 |
